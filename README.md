# BookNest Online Bookstore

This workspace contains a complete static bookstore website built with HTML, CSS, and JavaScript.

## Features

- Product catalog with categories and search
- Live filtering and sorting
- Shopping cart with quantity controls
- Persistent cart using `localStorage`
- Checkout form with order summary
- Responsive layout for desktop and mobile

## Files

- `index.html` – main website structure
- `styles.css` – responsive styling and layout
- `script.js` – product rendering, cart logic, filters, and checkout

## Run locally

1. Open `index.html` in your browser.
2. Or serve the folder using a local web server for a better experience.

## Customize

- Add more book entries in `script.js` inside the `products` array.
- Update styles in `styles.css`.
- Extend checkout behavior in `script.js`.
